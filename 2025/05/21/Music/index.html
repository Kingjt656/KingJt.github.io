音响硬件实习

一、功放板选购

功放板链接D800 -

功放板D1000

🔧 一、功能模块详解

1.1、主芯片（功放IC）

​ 以上的功放板是基于TPA3116D2的多功能数字功放板，具有多个音频输入方式和多种调节功能，适用于DIY音箱、家庭影院、K歌设备等场景。

①TPA3116D2 ×2（双通道）

  • 来自德州仪器（TI）的高性能D类数字功放芯片。
  • 支持双声道（立体声）或桥接模式，50W + 50W + 100W。
  • 效率高、发热低、失真小，适用于中小型功放应用。

1.2、 电源输入

  • 支持12V~24V直流电源输入
    • 适配不同功率需求的音响系统。
    • 输入电压越高，可驱动的扬声器功率越大。

1.3、音频输出接口

  • 左声道、右声道、低音炮输出端子（接喇叭）：

    • 实现2.1声道输出，带有独立的低音通道（Subwoofer）；

    其中2.1声道：2.1声道输出（左右声道+低音炮独立通道）

    • 适合中高频+低音分频的音响结构，提高音质表现。

1.4、音频输入方式：

  • USB输入：直接连接电脑或数字源播放；
  • AUX输入：3.5mm模拟音频输入；
  • 话筒模块接口：可连接麦克风用于K歌；
  • 引线式按键板接口：用于扩展遥控、模式切换等按键控制。

🎛️ 二、调节功能详解

2.1 功能旋钮（主板上）：

调节项 说明
低音炮频率（20-200Hz） 调整低音通道的分频点；滤掉中高频信号，只输出低频。
SUBWOOFER 音量调节 控制低音炮输出音量。
TREBLE（高音调节） 调节左右声道的高频增益，增强细节表现力。
BASS（低音调节） 调节左右声道的低频增益，增强低频厚度。
M-VOL（总音量+开关） 控制整机输出音量，并带电源开关功能。

2.2 外接麦克风模块调节：

接口 说明
麦克风接口D1/D2 插入话筒，进行语音输入或K歌；
麦克风风量调节 调整话筒接收灵敏度（电平）
麦克风音量调节 调整话筒声音在输出音频中的占比

🔌 三、接口一览与连接方式

名称 功能
左/右声道输出 接普通全频/高频喇叭
低音输出（Sub） 接低音喇叭
USB接口 音频输入接口（数字音频）
AUX接口 音频输入接口（模拟音频）
麦克风接口 D1/D2 音频输入（人声）
电源输入（DC 12~24V） 提供整板供电
控制板扩展口 扩展物理按键板（播放/暂停等）

📚 四、相关知识点总结

4.1 TPA3116D2 芯片知识

  • 属于D类（Class-D）数字功放；
  • 内建高效率开关电源转换器；
  • 优点：
    • 高效率
    • 体积小、发热低
    • 支持双通道模式

4.2 功放的基本概念

概念 说明
功放（Amplifier） 把微弱音频信号放大到足以推动扬声器的电平。
2.1声道 左声道+右声道+低音通道（独立Subwoofer）
分频器 将不同频率段信号分送给不同扬声器（如高频/低频）

4.3 音频参数调节基础

参数 说明
高音（Treble） 3KHz以上频率，提升清晰度和细节。
低音（Bass） 300Hz以下频率，提升沉浸感。
总音量（Master Volume） 控制整体输出电平。
频率调节（低音分频） 影响低音通道的响应范围，低于设定频率的声音才会输出到低音炮。

🔊 五、音频输出与功率参数

参数 含义与作用
声道数：2.1声道（50W + 50W + 100W） 2.1声道指的是： ① 左声道：最大50W ② 右声道：最大50W ③ 低音炮：最大100W 这种设计可以输出更加立体的音效，特别增强低频震撼感，适合观影和音乐。
适配喇叭 左右声道：20-50W、4-8Ω 低音炮：100W、4-8Ω ➡ 匹配范围比较广，可以兼容多数常见音响单元（例如6.5寸、8寸、10寸喇叭等）。（5 ~ 6.5寸低音喇叭）

1. 不同喇叭类型特点比较

类型 特点 适用场景
全频喇叭（如15W款） 可涵盖大部分频段（低中高） 小音箱/广播/简单音乐播放
中高音喇叭（如10W款） 强调高频细节，缺低音 高频补偿、报警提示、高音箱

2. 功放芯片与喇叭匹配原则（以TPA3116D2为例）

匹配要点 推荐范围 错误搭配后果
阻抗匹配 4Ω >30Ω 时输出功率锐减，音量小
功率匹配 喇叭额定功率 ≈ 功放最大输出功率 功率不匹配易烧喇叭或音量不足
灵敏度选择 ≥85dB 低灵敏度喇叭声音小，需更大功率
频响适配 与应用场景匹配（如人声推荐120Hz~15kHz） 高音不足或低音缺失

4. 核心参数对声音的影响

参数 影响分析
尺寸（口径） 决定喇叭推动空气量的能力。- 大尺寸（如6.5寸）→ 低频强、下潜深，有力感。- 小尺寸（如2.5寸）→ 高频响应更快，适合播放人声细节和高频延伸。
额定功率 表示喇叭长期工作的最大安全功率。- 功率越大，音响的动态余量越充足，爆发力更好。
阻抗（4Ω） 影响与功放的匹配。- 通常4Ω阻抗更容易被驱动，适合家用功放。- 多喇叭串联/并联时需注意总阻抗匹配。
频率响应 描述喇叭能发出声音的频率范围。- 响应范围越宽，声音层次越丰富。- 低频起点低（如51Hz）→ 低音更沉稳。- 高频终点高（如12kHz）→ 更有空气感、清晰度高。
灵敏度（dB） 表示喇叭发出相同音量所需输入功率的大小。- 数值越高，越容易出声（效率高）。- 82dB属中等偏低，适合安静环境；85dB以上则更省力。
谐波失真（THD） 表示喇叭播放音频时的失真程度。- 5%为中等水平。- 失真越小，声音越纯净。
磁路结构 决定磁场强度和控制力。- 外磁式：结构简单，散热好，广泛用于家用音箱。- 内磁双磁：磁力集中，体积小，控制力更强，适合精细高频。
音圈尺寸 影响喇叭的功率承载与控制力。- 音圈越大→ 耐热能力强，控制低频更好。- 25芯适合高功率、大动态需求。
重量 一定程度反映磁体体积与用料。- 重量大，低频更有力，结构更稳固。- 太轻则音色可能发飘或单薄。
